## リスト操作: 条件付き変換
### 問題
リスト内の正の整数を条件に基づき変換する関数をデバッグします。正の偶数はそのまま、正の奇数は3倍にし、負の数、0、非整数は無視する仕様です。
### あなたの回答
```python
def main(numbers):
    result = []
    for num in numbers:
        if isinstance(num, int):
            if num > 0 and num % 2 == 0:
                result.append(num)
            elif num > 0 and num % 2 != 0:
                result.append(num * 3)
    return result
```
