## 1からnまでの整数の合計
### 問題
整数 n を入力として受け取り、1からnまでの整数の合計を計算して返す関数を作成します。n が0の場合は 0 を返すことに注意してください。
### あなたの回答
```python
def main(n):
    sum = 0
    for i in range(n+1):
        sum += i
    return sum
```
